Nmap Dashboard

Prerequisites

Before you can run the Nmap Dashboard, you need to have the following installed and configured:

  • Node.js and npm: Node.js is required to run the server. npm is used to manage dependencies.
  • Nmap: The Nmap tool must be installed to perform network scans.

Installation Steps

1. Install Node.js and npm

Download and install Node.js from the official Node.js website. This will also install npm.


# For Windows and macOS:
# Download the installer from the official website and follow the instructions

# For Linux:
sudo apt update
sudo apt install nodejs npm

2. Install Nmap

Install Nmap according to your operating system:


# For Windows:
# Download the installer from the Nmap official website and follow the instructions

# For macOS (using Homebrew):
brew install nmap

# For Linux (Debian-based distributions):
sudo apt update
sudo apt install nmap

3. Set Up the Project

Clone or download the project, then install the necessary dependencies.


# Clone the repository
git clone https://github.com/hareram1/Nmap_GUI.git
cd your-repository

# Install dependencies
npm install

4. Run the Server

Start the server with Node.js.


node server.js

The server will be running on http://localhost:3000. Open this URL in your web browser to use the Nmap Dashboard.
